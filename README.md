# Pranathi Deepak Portfolio

Interactive React + TailwindCSS portfolio for recruiters reviewing Pranathi Deepak's machine learning, GenAI, RAG, and MLOps experience.

## Run Locally

This scaffold is intentionally zero-install. It vendors React, ReactDOM, and the Tailwind browser runtime so GitHub Pages can serve it directly.

```bash
node scripts/serve.mjs
```

Then open `http://localhost:4173`.

## Edit Content

Most portfolio content lives in `src/data.js`:

- Contact links
- Resume summary
- Recruiter highlights
- Skills and filters
- Experience timeline
- Projects
- Education

The UI lives in `src/App.js`, and custom polish beyond Tailwind lives in `src/styles.css`.
